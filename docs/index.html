üí° How It Works

üìÇ File Organization

Same base filename = one activity

Files like run1.csv, run1.tcx, run1.fit are merged into a single JSON activity object.

Different filenames = separate activities

Upload run1.* and run2.* to create 2 distinct activity objects in the output.

üìä File Types

TCX files (recommended)

Contain precise timestamps, lap data, trackpoints, and heart rate information. Best for complete activity data.

CSV files

Typically contain split summaries and aggregated statistics from your watch.

FIT files

Binary format - basic parsing only in browser. Use the CLI tool for full FIT parsing.
